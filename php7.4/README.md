
# README ENV

Install and setup the instance
---------------------------------------
`sh install.sh`

## Adding a new user
---------------------------------------
`add-user.sh USERNAME PASSWORD`

## Remove user (and all its files)
---------------------------------------
`remove-user.sh USERNAME`

## Adding a new site
---------------------------------------
`add-site.sh USERNAME DOMAIN`

## Remove site (and all its files)
---------------------------------------
`remove-site.sh USERNAME DOMAIN`
